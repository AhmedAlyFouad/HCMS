## Description
HCMS API is a FastAPI-based backend system designed to manage healthcare complaints.
It provides secure user authentication, complaint submission, status tracking, statistics, and comment management using JWT authentication and SQL Server.

This project is suitable for healthcare platforms, government services, or enterprise complaint management systems.

---

## Features
- User registration & login with JWT authentication
- Secure password hashing using bcrypt
- Role-based user handling
- Create, view, and resolve complaints
- Complaint statistics (total, pending, solved, unsolved)
- Comment system for complaints
- SQL Server database integration
- Environment-based configuration using `.env`

---

## Tech Stack
- **Backend:** FastAPI
- **Language:** Python
- **Authentication:** OAuth2 + JWT
- **Database:** SQL Server (pyodbc)
- **Security:** bcrypt, python-jose
- **Environment Management:** python-dotenv
